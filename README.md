# 2D-Platformer
산나비라는 게임을 보고 마우스 클릭을 통한 와이어 이동 기능과 대시를 이용하여 빠른 이동을 통해 클리어 지점에 도착하면 클리어 되는 게임. 동영상은 루트 폴더의 Movie02.mp4 입니다.

# 사용한 기능
 - Rigidbody2D, Coroutine 을 사용하여 A, D 키를 두 번 연속 누를 시 빠르게 이동하는 대시 기능 구현
 - Raycast, Distance Joint 2D, Line Renderer 를 사용하여 마우스 클릭 시 플레이어, 벽 사이에 선을 연결해서 가속도를 주고 연결된 선 길이로 고정된 원형 이동 기능 구현
 - Queue, Coroutine 을 사용하여 플레이어의 현재 이동속도가 기본 이동속도보다 빨라졌을 경우 잔상 효과를 추가하여 속도감을 느낄 수 있도록 함

# 추가하고 싶은 기능
 - 플레이어의 이동, 대시, 와이어에 맞는 2D Asset 을 찾아서 애니메이션 추가
 - 여러 스테이지를 추가하여 레벨 디자인에 좀 더 신경 써서 낙사, 이동속도 관련 아이템, 체크 포인트 등을 추가
 - 몬스터도 추가하여 기본 공격, 와이어를 통한 공격 등 전투 추가

# 어렵다고 느낀 부분
 - 아직 게임 개발과 전체적인 Unity 의 코드 흐름에 익숙치 않다보니 버그가 발생하거나 전혀 예상치 못한 동작이 생겼을 때 이걸 어떻게 디버깅 해야할지 되게 막막했음. 디버깅 관련 공부를 잘 해야할 거 같다고 느꼈음. 특히 AI 를 사용해도 거의 의미가 없는 영역이라고 생각함.
